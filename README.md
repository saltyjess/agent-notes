# Agent Notes

A settings UI for managing itinerary notes, built with React, TypeScript, and Tailwind CSS.

## Live Demo

**https://saltyjess.github.io/agent-notes**

## Features

- **Vendor Notes** - Notes displayed below vendor information on itineraries
- **Trip Notes** - Notes shown in the "Before you go" section
- **Company Notes** - Company-wide notes with toggle controls

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS v4
- Radix UI (Switch, Dropdown Menu)
- Phosphor Icons

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```

## Design

Based on the Nomad Design System from Figma.
